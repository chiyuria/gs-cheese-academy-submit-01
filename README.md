# ⑤工夫した点・こだわった点

* 冒頭で“チーズが溶けるような”イントロ演出を入れ、最初の印象を強化
* スクロール時にふわっと要素が現れるアニメーションを実装し、心地よい閲覧体験を設計
* アニメーションの秒数・delay を細かく調整し、全体のテンポを最適化
* レイアウトやパーツ構成は仕様書に忠実に再現
* 特にフォーム周辺は、HTMLの構造を意識しながら階層整理して構築した

# ⑥難しかった点・次回トライしたいこと（又は機能）

* フォームのレイアウト調整が最難関で、label と input の配置やチェックボックス群をピクセル単位で微調整する必要があった
* legend を非表示にしたり、z-index の重なりを管理したりと、アニメーションとの両立にも苦戦
* 次回はレスポンシブ対応に挑戦したい

  * 今回はPC表示を優先したため、スマホでも快適に閲覧できる設計を練習したい
